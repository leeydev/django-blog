# Django Blog

Full-featured blog application built with Django.

## Features

- 📝 Post CRUD operations
- 👤 User authentication
- 🔍 Search functionality
- 📊 Admin interface
- 🎨 Template system

## Installation

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Usage

Visit http://localhost:8000
